# Project 2 - Backend API Development

## Overview

This project focuses on building a simple backend API using Node.js and Express. The API handles user requests, validates input, and returns JSON responses.

## Features

* REST API Development
* GET Endpoint
* POST Endpoint
* Input Validation
* JSON Request and Response Handling

## Technologies Used

* Node.js
* Express.js

## API Endpoints

### GET /users

Returns all users.

### POST /users

Adds a new user.

Request Body:
{
"name": "Alok"
}

## Installation

1. Install dependencies:
   npm install

2. Start the server:
   node server.js

3. Open:
   http://localhost:3000
